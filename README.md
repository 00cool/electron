# Atom Shell

Experimental native layer for the [Atom](https://github.com/github/atom).

## Development

### One-time setup

    $ script/bootstrap https://gh-contractor-zcbenz.s3.amazonaws.com/libchromiumcontent

### Building

    $ script/build

This will build the app into the `build` directory.
